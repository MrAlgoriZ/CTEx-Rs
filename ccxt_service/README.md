# CPS (ccxt-python-service) - микросервис для проекта CTEx-Ai

Данный сервис разворачивает бекенд на порте 3737, сам сервис нужен для интеграции с ccxt из другого контейнера / кода. 

## Запуск:
```bash
chmod +x ./start.sh
./start.sh
```

## Зависимости

### Системные
uv

### Для Python
- ccxt>=4.5.34
- fastapi>=0.128.0
- uvicorn>=0.40.0
